#LinkIT jQuery Plugin#

**LinkIn** is a simple jQuery plugin to attach links to elements

##Version##

0.3.0

##Usage##
	    $('span').linkIt({
          href: 'http://test.com',
          text: 'click me',
          target: '_blank'
        });


##Vendors##

jQuery - [http://jquery.com](http://jquery.com)
